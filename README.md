# DevToys.HammingDistance

Hamming Distance Calculator for [DevToys](https://devtoys.app/) 2.0.

## Installation

1. Download the `DevToys.HammingDistance` NuGet package from [NuGet.org](https://www.nuget.org/packages/DevToys.HammingDistance).
1. In DevToys, open `Manager Extensions`, click on `Install` and select the downloaded NuGet package.
